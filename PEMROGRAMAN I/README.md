Untuk kompilasi file Java dari terminal Windows:

1. Buka terminal di folder yang berisi file `.java`.
2. Jalankan:
   - `javac NamaFile.java`

Contoh:

- Jika berada di dalam `...\PEMROGRAMAN I\P3_HAPIZDNURYADIN\source package\package\latihan` dan ingin kompilasi `FormatAngka.java`:
  - `javac FormatAngka.java`

Untuk menjalankan program Java setelah dikompilasi:

- `java NamaKelas.java`

Contoh:

- Setelah kompilasi `FormatAngka.java`, Untuk menjalankan:
  - `java FormatAngka.java`


Catatan:
- Pastikan `javac` dan `java` sudah tersedia di PATH.
